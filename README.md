Leads Tracker

A simple Leads Tracker web app built as part of the Scrimba Frontend Developer Career Path. The application allows users to quickly save and manage useful links (leads) directly from their browser.

Features
Save URLs manually
Save the current browser tab with one click
Store leads using localStorage so they persist between sessions
Delete all saved leads
Clean and responsive UI

Tech Stack
HTML
CSS
JavaScript
Chrome Extension APIs
LocalStorage

What I Learned
How to build a Chrome browser extension
Working with the Chrome Tabs API
Using localStorage for persistent data
Handling DOM manipulation and event listeners in JavaScript
